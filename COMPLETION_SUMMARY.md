## 🎯 项目概述

@ldesign/engine 是一个现代化、模块化的跨框架应用引擎，提供统一的插件系统、状态管理、事件系统等核心功能，支持 Vue 3、React、Angular、Svelte、Solid.js 等主流前端框架。

## 💡 设计决策

### 1. 三层架构
采用核心层、适配器层、框架层的设计，实现框架无关的核心功能。

### 2. 插件工厂函数
使用工厂函数创建插件，提供更好的类型推断和配置清晰度。

### 3. 框架特定包
为每个框架创建独立的适配器包，支持按需安装和更好的 Tree-shaking。

### 4. 统一的 API
所有框架使用统一的插件 API，降低学习成本。
